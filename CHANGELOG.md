# Changelog

## [1.1.0] - 2026-03-26

### Added
- **多视角取景框 (Perspective Frames)**：第二步新增第5种方法论，要求从至少3个不同利益相关者视角审视同一问题
- **奥卡姆剃刀过滤**：第三步本质重构阶段新增显式过滤标准——核心假设数量最少，但解释力最强
- **贝叶斯校验 (Bayesian Check)**：新增第四步，要求标注结论的置信度、关键脆弱点、以及什么新证据会推翻当前结论

### Changed
- 更新 description 以反映完整的四步流程

## [1.0.0] - 2026-03-26

### Added
- 初始版本
- 四种批判性检验方法论：笛卡尔怀疑法、苏格拉底式提问、功能主义拆解、反向思维
- 三步分析流程：直觉初稿 → 批判检验 → 本质重构
