# First-Principle Analyst

针对复杂问题、网页或文章进行深度第一性原理分析的 AI Skill。

## 安装

将本仓库克隆到 deepagents skills 目录：

```bash
git clone https://github.com/senmud/first-principle-analyst.git ~/.deepagents/agent/skills/first-principle-analyst
```

## 使用

安装后，在 deepagents 会话中触发即可。适用于：

- 分析公众号文章、技术博客、行业报告
- 拆解商业决策、技术方案、投资逻辑
- 任何需要"看透本质"的复杂问题

## 分析流程

```
第一步：直觉式初稿
  ↓
第二步：批判性检验（5种方法论）
  ├── 笛卡尔怀疑法
  ├── 苏格拉底式提问
  ├── 功能主义拆解
  ├── 反向思维
  └── 多视角取景框
  ↓
第三步：本质重构（奥卡姆剃刀过滤）
  ↓
第四步：贝叶斯校验（置信度 & 脆弱点）
  ↓
输出：直击本质的深度报告
```

## 兼容性

- deepagents-cli
- 任何支持 SKILL.md 格式的 AI Agent 框架
